# MkDocs

MkDocs是一个静态站点生成器，专注于构建项目文档，使用Markdown编写文档源文件，并通过单个YAML配置文件进行配置。MkDocs是一个**快速**，**简单**，**极具美感**的工具。您可以首先阅读[介绍性教程]，然后查看[用户指南]获取更多信息。

[介绍性教程]: getting-started.md
[用户指南]: user-guide/README.md
