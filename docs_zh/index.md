# MkDocs

使用Markdown构建项目文档。

---

MkDocs是一个**快速**、**简单**且**美观**的静态网站生成器，适用于构建项目文档。文档源文件采用Markdown编写，并使用单个YAML配置文件进行配置。首先阅读[入门教程]，然后查看[用户指南]获取更多信息。

[入门教程]: getting-started.md
[用户指南]: user-guide/README.md
