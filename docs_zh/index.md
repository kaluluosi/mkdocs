# MkDocs

用&nbsp;Markdown 编写项目文档。

---

MkDocs是一个**快速**、**简单**又**十分漂亮**的静态网站生成器，专注于构建项目文档。文档源文件采用Markdown编写，并配以单个YAML配置文件。请先阅读[入门教程]，然后查看[用户指南]以获取更多信息。

[入门教程]: getting-started.md
[用户指南]: user-guide/README.md
